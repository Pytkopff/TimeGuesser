# MÓJ PROFIL TECHNOLOGICZNY (BASE BUILDER)
Jestem developerem budującym aplikację Farcaster Frame v2 na blockchainie Base.

Mój Tech Stack (to co działało w poprzednim projekcie):
- Framework: Next.js (App Router)
- Frame Library: Frog.fm lub OnchainKit (wybierzemy lepsze do v2)
- Database: Supabase (Postgres) - kluczowe do zapisu wyników
- Stylizacja: TailwindCSS (Mobile-first)
- Język: TypeScript (strict mode)

Zasady projektu:
1. DESIGN: Aplikacja musi wyglądać dobrze na telefonie (pionowo).
2. UX: Proste, duże elementy (jak w Chronophoto, ale mobile).
3. CEL: Zbieranie transakcji on-chain (Builder Score).
